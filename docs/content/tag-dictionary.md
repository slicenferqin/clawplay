# ClawPlay 标签词表（第一版）

- Last Updated: 2026-03-08
- Scope: 投稿软约束、后台审核提示、后续筛选与聚合

## 1. 目标

标签不是“随手发挥”的装饰，而是后续搜索、合集、相关推荐和热门归档的基础索引。

第一版词表只做三件事：

1. 给投稿人一份可以直接复用的推荐标签。
2. 把高频同义词收敛到少量标准词。
3. 先做建议与提示，不做强制拦截。

## 2. 使用规则

- 每个 Soul 建议 3-6 个标签
- 优先写“能力 / 任务 / 场景”，少写空泛人设形容词
- 同一份 Soul 不要同时写多个同义标签
- 如果必须使用新标签，优先保证它可长期复用，而不是一次性梗标签

## 3. 推荐标签

### 工作与决策

- `规划`
- `策略`
- `项目管理`
- `产品分析`
- `复盘`

### 开发与工程

- `开发`
- `架构`
- `系统设计`
- `调试`
- `代码审查`
- `测试`
- `文档整理`

### 学习与表达

- `学习`
- `提问`
- `思辨`
- `翻译`
- `写作`
- `创意写作`
- `技术讲解`

### 角色与互动

- `陪伴`
- `角色扮演`

## 4. 同义词收敛建议

| 非推荐写法 | 建议统一为 |
| --- | --- |
| `计划` / `计划制定` | `规划` |
| `审议` | `策略` |
| `优先级` / `敏捷` | `项目管理` |
| `编程` / `编程搭档` / `coding` | `开发` |
| `debug` | `调试` |
| `审查代码` | `代码审查` |
| `回归` | `测试` |
| `引导` | `提问` |
| `陪聊` | `陪伴` |
| `轻技术` | `技术讲解` |
| `趣味化` / `海盗` | `角色扮演` |

## 5. 现阶段允许保留的自由标签

以下类型暂时允许保留，不做强制收敛：

- 角色 IP，例如 `猫娘`
- 语气标签，例如 `毒舌`
- 明确且可复用的垂直标签，例如 `扩展性`

但这些标签更适合作为补充标签，而不是唯一的检索入口。

## 6. 维护原则

- 只有在站内连续出现、且具备可复用价值的新标签，才进入推荐词表
- 同义词映射优先解决“会把搜索打散”的高频写法
- 每次新增推荐词，应同步更新：
  - `src/lib/content-rules.ts`
  - 本文档
  - 投稿端推荐标签 UI
