# 🦞 ClawPlay

> OpenClaw 中文 SOUL.md 分享社区 - 让你的 AI 助手更有灵魂

ClawPlay 是一个专注于中文的 OpenClaw SOUL.md 人格配置分享平台。在这里，你可以发现、分享和定制各种有趣的 AI 人格，让你的 OpenClaw 助手更符合你的需求。

## 🌟 什么是 SOUL.md？

SOUL.md 是 OpenClaw 的人格配置文件，它定义了你的 AI 助手的：
- 性格特征和说话风格
- 专业领域和技能倾向
- 工作方式和互动偏好
- 价值观和行为准则

通过替换 SOUL.md，你可以让同一个 AI 助手展现完全不同的个性和能力。

## 🚀 快速开始

### 使用现有人格

1. 浏览上面的分类，找到感兴趣的人格
2. 点击链接查看详细配置
3. 复制 SOUL.md 内容
4. 替换你的 OpenClaw 配置目录中的 `SOUL.md` 文件
5. 重启 OpenClaw，享受新人格！

### 贡献你的人格

我们欢迎所有原创和翻译的优质人格配置！

1. Fork 本仓库
2. 在对应分类下创建你的 SOUL.md 文件
3. 填写人格说明（使用模板）
4. 提交 Pull Request

详见 [贡献指南](./CONTRIBUTING.md)

## 📖 人格模板

每个人格配置应包含：

```markdown
# 人格名称

## 简介
一句话描述这个人格的特点

## 适用场景
- 场景1
- 场景2

## 特色功能
- 功能1
- 功能2

## 使用建议
给用户的使用提示

## 作者信息
- 作者：xxx
- 来源：原创 / 翻译自 [链接]
- 协议：MIT / CC BY 4.0

---

## SOUL.md 内容

\`\`\`markdown
[这里是完整的 SOUL.md 配置]
\`\`\`
```

## 🤝 贡献规范

- ✅ 原创内容：鼓励分享你调教的独特人格
- ✅ 翻译内容：必须标注原作者和来源链接
- ✅ 开源协议：建议使用 MIT 或 CC BY 4.0
- ❌ 禁止内容：违法、暴力、歧视性内容
- ❌ 版权侵犯：未经授权的商业人格配置
