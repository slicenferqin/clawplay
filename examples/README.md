## 💡 最佳实践

### 设计原则
1. **明确定位**：清楚这个人格解决什么问题
2. **个性鲜明**：有独特的说话风格和行为模式
3. **实用为主**：不要过度追求"有趣"而忽略实用性
4. **适度约束**：给 AI 明确的边界和行为准则

### 常见问题

**Q: SOUL.md 应该多长？**
A: 没有固定长度，但建议：
- 核心部分：500-1000 字
- 包含示例：1000-2000 字
- 过长可能影响效果，过短可能不够具体

**Q: 如何测试人格效果？**
A:
1. 准备几个典型场景的测试对话
2. 观察 AI 的回应是否符合人格设定
3. 多次测试，看一致性如何
4. 让其他人试用并收集反馈

**Q: 可以混合多个人格吗？**
A: 可以，但要注意：
- 确保不同特质之间不冲突
- 保持核心定位清晰
- 避免人格过于复杂
